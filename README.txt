1.Git is a tool that is used to manage ,control and lets you keep a track of your source code which are then transfered into a git repository
while Github is a platform or a type of location for uploading copies of your source code in a git repository.

2.five basic commands are:-
git clone     Clone a repository into a new directory
git add       Add file contents to the index
git status    Show the working tree status
git merge     Join two or more development histories together
git commit    Record changes to the repository
